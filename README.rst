Python
==================
python 有三種資料型態（Type）
- 整數(int)           123
- 字串(str)          ‘123’ 
- 浮點數(float)      0.123   

不同型態的運算要注意，把字串和數值相加，<br/>
會出現錯誤（TypeError），<br/>
如果要運算不同類型的型態，可以轉變型態，<br/>

型態轉變舉例：
a str(123)
b int(123)
